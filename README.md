> Minimalist setup for SwayWM. 🚀

![Desktop Preview](./preview.png)

### 🛠️ Quick Info
- **WM:** SwayFX
- **Bar:** Waybar
- **Terminal:** Foot
- **Shell:** Zsh / Oh-my-zsh
- **File Manager:** Yazi / Nemo

### 📦 Installation
-clone the repo
-git clone [https://github.com/hochazxz/desktop.git](https://github.com/hochazxz/desktop.git)
-backup all your files
-Copy all files in dotfiles and paste to your $HOME folder
> cp -rfT dotfiles/dotfiles $HOME/
-other setup
> chsh -s /usr/bin/zsh
systemctl --user enable pipewire pipewire-pulse wireplumber
